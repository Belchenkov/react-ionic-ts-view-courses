# View Courses (React/Ionic/TS)

> Build App "View Courses" usage React + Ionic 5 + Typescript

#### Get Started
```sh
   git clone 
```
```sh
   ionic serve 
```
or 
```sh
   npm start
```

> Run on Android
```sh
   ionic capacitor run android
```

> Run on IOS
```sh
   ionic capacitor run ios
```
